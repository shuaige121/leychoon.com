# Ley Choon Group Website Redesign (项目重构计划)

## 项目简介 (Project Overview)
本项目旨在全面重构 Ley Choon Group 的官方网站，以适应新加坡的多元化市场需求。主要目标是实现多语言支持，并优化移动端的联系方式，提供更现代化的用户体验。

## 主要功能 (Key Features)

### 1. 多语言支持 (Multilingual Support)
网站将支持以下四种语言的无缝切换，以服务不同族群的用户：
- **英文 (English)** - 默认语言
- **中文 (Chinese)**
- **马来文 (Malay)**
- **淡米尔文 (Tamil)**

### 2. 便捷联系方式 (Enhanced Connectivity)
针对移动端用户习惯进行深度优化：
- **WhatsApp 集成**: 所有的手机号码将转换为 WhatsApp 链接，点击即可直接发起对话。
- **一键邮件 (Direct Email)**: 邮箱地址将配置为 `mailto` 链接，点击即可调用默认邮件客户端发送邮件。

### 3. 现代化设计 (Modern Design)
- **响应式布局**: 确保在手机、平板和电脑上都能完美展示。
- **视觉升级**: 采用符合现代审美的高端设计风格。

## 技术栈 (Tech Stack)
- **前端核心**: HTML5, CSS3, JavaScript (Vanilla/ES6+)
- **样式框架**: 自定义 CSS (Vanilla CSS) 以确保灵活性和轻量化
- **部署**: 静态页面部署 (如 Cloudflare Pages / GitHub Pages)
